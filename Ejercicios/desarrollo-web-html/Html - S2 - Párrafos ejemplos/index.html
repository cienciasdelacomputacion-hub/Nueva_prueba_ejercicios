Ejemplo de párrafos en HTML

Este es un párrafo simple. Los párrafos en HTML se escriben entre las etiquetas <p> y </p>.

Cada párrafo crea un bloque separado de texto, lo que significa que el contenido se muestra uno debajo del otro con un pequeño espacio entre ellos.

Los párrafos se usan para escribir textos descriptivos, explicativos o narrativos. También podés incluir negritas, cursivas o enlaces dentro del mismo.

HTML ignora los espacios en blanco múltiples y los saltos de línea dentro de un párrafo, por eso si querés forzar un salto, necesitás usar otras etiquetas como <br>.
