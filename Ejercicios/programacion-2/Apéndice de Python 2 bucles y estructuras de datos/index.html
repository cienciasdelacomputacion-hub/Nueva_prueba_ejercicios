Piensa en las listas como una hilera de casilleros, cada uno guardando un objeto. Son colecciones **ordenadas** (mantienen el orden de inserci√≥n) y **mutables** (puedes cambiar sus elementos).

  • üìå Admiten duplicados.
  • üî¢ Acceso por √≠ndice: siempre empiezan en 0.

Los bucles son esenciales para **repetir acciones** eficientemente. Puedes recorrer elementos, contar o esperar a que una condici√≥n se cumpla.

  • for: Recorre cada elemento de una secuencia (como una lista o un string) directamente.
  • for con range(): Es √∫til para iterar un n√∫mero espec√≠fico de veces o por √≠ndices.
  • while: Repite un bloque de c√≥digo **mientras** una condici√≥n sea verdadera.
# for: Recorre una lista
for fruta in ["manzana", "banana", "kiwi"]:
    print(fruta)

# for con range(): Cuenta hasta 5
for i in range(5):
    print("N√∫mero:", i)

# while: Repite hasta que se cumpla la condici√≥n
contador = 0
while contador < 3:
    print("Repetici√≥n:", contador)
    contador += 1

Un **diccionario** es una colecci√≥n desordenada que almacena datos en pares **clave-valor**. Es como una gu√≠a telef√≥nica: usas un nombre (la clave) para encontrar un n√∫mero (el valor).

persona = {"nombre": "Ana", "edad": 25, "ciudad": "Montevideo"}

# Acceder a un valor
print(persona["nombre"]) # Salida: 'Ana'

# Modificar o agregar un par
persona["edad"] = 26
persona["profesion"] = "Desarrolladora"

# Recorrer un diccionario
for clave, valor in persona.items():
    print(f"'{clave}' es '{valor}'")

Podemos usarlos dentro de listas para representar colecciones de objetos complejos:

    {"id": 2, "nombre": "Pedro"}

Las **secuencias** (listas, strings, tuplas) tienen funciones que facilitan su manejo. Aqu√≠ algunas de las m√°s √∫tiles:

  • len(): Obtiene la longitud de la secuencia.
  • min() y max(): Encuentran el valor m√≠nimo y m√°ximo.
  • sorted(): Devuelve una nueva lista con los elementos ordenados.
  • Operador in: Verifica si un elemento pertenece a la secuencia.

print(len(data))    # Salida: 4
print(max(data))    # Salida: 9
print(sorted(data)) # Salida: [1, 3, 5, 9]

if 5 in data:
    print("El 5 est√° en la lista")

Las **comprensiones de listas** son una forma concisa de crear listas nuevas a partir de una existente. Es la forma "Python" de mapear y filtrar.

  • Mapeo: Transforma cada elemento de una lista. La sintaxis es [expresi√≥n for item in iterable].
  • Filtrado: Selecciona solo los elementos que cumplen una condici√≥n. La sintaxis es [expresi√≥n for item in iterable if condici√≥n].
numeros = [1, 2, 3, 4, 5, 6]

# Filtrado: obtener solo los n√∫meros pares
pares = [x for x in numeros if x % 2 == 0]
print(pares) # Salida: [2, 4, 6]

# Mapeo: obtener el cuadrado de cada n√∫mero
cuadrados = [x**2 for x in numeros]
print(cuadrados) # Salida: [1, 4, 9, 16, 25, 36]
