Entendiendo la Propiedad `display` en CSS

La propiedad display es fundamental para controlar cómo se muestra y se comporta un elemento en el layout de la página. Observa las diferencias entre los tipos principales.

Display por Defecto: `block`

Los elementos block siempre inician en una nueva línea y, por defecto, ocupan todo el ancho disponible de su contenedor.

Este es un DIV (Block por defecto).
Este es otro DIV (Block por defecto).

Este es un Párrafo (P, también Block por defecto).

Display por Defecto: `inline`

Los elementos inline no inician en una nueva línea y solo ocupan el ancho necesario para su contenido. No respetan width ni height explícitos y el margin/padding vertical puede afectar la línea de texto.

Este es un SPAN (Inline por defecto). Este es otro SPAN (Inline por defecto). Este es un Enlace A (Inline por defecto). Texto que continúa el flujo.

Cambiando Display: `inline-block`

Los elementos con display: inline-block; se comportan como elementos inline (no fuerzan nueva línea), pero respetan width, height, y todos los márgenes/rellenos, como si fueran bloques. Ideal para elementos de cuadrícula simples o botones con dimensiones.

DIV con display: inline-block;
Otro DIV con display: inline-block;
SPAN con display: inline-block;

Cambiando Display: `block` a Inline y `inline` a Block

Podemos cambiar el comportamiento por defecto si lo necesitamos.

Este DIV (normalmente Block) ahora es display: inline;.
Texto después del DIV inline. Este SPAN (normalmente Inline) ahora es display: block;. Texto después del SPAN block.

Ambos ocultan elementos, pero de forma diferente.
