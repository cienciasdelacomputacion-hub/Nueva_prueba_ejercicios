Este párrafo está dentro de una caja con un color de fondo.

Explicación del Ejemplo:

Aquí, seleccionamos el div con la clase caja-con-fondo y le aplicamos

background-color:#e0f7fa;

Esto hace que todo el espacio ocupado por el div, incluyendo el relleno de 20px, se pinte de ese color azul verdoso claro.

El borde también se dibuja sobre este color de fondo.

El párrafo dentro hereda propiedades o tiene las suyas propias (como el color de texto), pero el fondo es una propiedad del div contenedor.
