Ejemplos de Alineación de Texto

Alineado a la Izquierda

Este texto está alineado a la izquierda. Esta es la alineación por defecto y generalmente la mejor para bloques largos de lectura en idiomas que se leen de izquierda a derecha.

Centrado

Este texto está centrado. Es útil para títulos o citas cortas, pero no recomendado para párrafos largos.

Alineado a la Derecha

Este texto está alineado a la derecha. Se usa menos comúnmente en layouts de izquierda a derecha, a menudo para elementos específicos o diseños creativos.

Justificado

Este texto está justificado. Se ajusta el espacio entre palabras para que ambos bordes, izquierdo y derecho, estén alineados. Esto puede crear un bloque visualmente uniforme, pero a veces introduce espacios grandes entre palabras si el texto no llena la línea.
